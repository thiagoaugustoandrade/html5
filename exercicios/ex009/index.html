Tipos de Lista

Existem 4 tipos de Lista que podem ser criadas por HTML:

1 - Listas Ordenas

Listas Ordenadas podem ser concretizadas pelas tags "ol" (order list) e seus itens com a tag "li". E só podem conter de quatro tipos - "A", "a", "I", "i".

  1. Malkuth
  2. Yesod
  3. Hod
  4. Netzach
  5. Tipheret
  6. Geburah
  7. Chesed
  8. Binah
  9. Chohmah
  10. Kether

2 - Listas Não-Ordenadas

Listas Não-Ordenadas podem ser concretizadas pelas tags "ul" (Unorder List) e seus complementos com a tag "li". Podem conter 4 tipos - "disc", "circle" ou "square". Veja os seguintes exemplos.

  • Pão
  • Leite
  • Café
  • Presunto
  • Queijo
  • Cheddar
  • Nescau

4- Lista de Definições

Listas de Definições servem para definir termos com precisão. Elas são escritas com as tags "dl"(definition list) seguidas tags "dt"(definition terms) e "dd" (device definition) Ex:

    HTML
    É uma tecnologia de desenvolvimento de conteúdo do site
    CSS
    É uma tecnologia de desenvolvimento de design de sites
    JS
    JavaScript é uma linguagem de programação para interação de sites
